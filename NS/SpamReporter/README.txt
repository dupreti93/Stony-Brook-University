1. Python version:
	3.0

2. How to run:

	i. Place the spam mbox file with name ‘mailbox.mbox’ in the main directory of project folder

	ii. In terminal, navigate to project folder and type:

			sudo python spamReporter.py
	

	A PDF file will open displaying summary of spam mailbox. This pdf can later be accessed from the project's main directory as 'result.pdf'.

Libraries required:

1. mailbox
2. fpdf
3. re
4. socket
5. pygeoip
6. numpy
7. matplotlib.pyplot
8. PyPDF2
9. webbrowser
